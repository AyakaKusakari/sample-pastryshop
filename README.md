# WordPressの自作テーマ（企業サイト）

WordPressの練習として、自作テーマで架空の洋菓子店のサイトを作成しました。

[»「架空の洋菓子店のサイト」へのリンク](https://wayasblog.work/sample-pastryshop/)

## ポイント

- デザイン制作・コーディング・WordPress化を実施 
- 「News」「About」「Menu」「Access」ページを固定ページで実装
- トップページに「News」の新着情報が出るように実装
- 「News」の個別ページの下に関連記事が出るように実装
- 「drawer.js」を実装 
- 「Contact Form 7」でお問合せフォームを実装 
 
 ## 使用言語
 
 HTML5 / CSS3 / Sass / jQuery / PHP
